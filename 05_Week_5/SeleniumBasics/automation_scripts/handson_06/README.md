# Hands-On 6: Selenium Test Automation with Pytest

## Objective

The objective of this hands-on exercise is to build a structured Selenium automation project using Pytest. This exercise focuses on writing reusable test cases, implementing fixtures, parameterization, HTML reporting, and capturing screenshots for failed test executions.

---

## Topics Covered

- Selenium WebDriver with Pytest
- Pytest Fixtures
- Test Parameterization
- Test Assertions
- HTML Test Reports
- Screenshot Capture
- Selenium Automation Best Practices

---

## Tasks Completed

- Configured Selenium WebDriver with Pytest.
- Created reusable fixtures for browser setup and teardown.
- Automated Selenium Playground test scenarios.
- Executed parameterized test cases with multiple input values.
- Implemented assertions to validate application behavior.
- Generated HTML execution reports.
- Captured screenshots automatically for failed test cases.
- Organized the project into a maintainable folder structure.
