# Hands-On 5: Selenium Locators and Waits

## Objective

The objective of this hands-on exercise is to understand different Selenium locator strategies and synchronization techniques. This exercise focuses on identifying web elements using various locators and implementing waits to improve the reliability of Selenium automation scripts.

---

## Topics Covered

- Selenium Locators
  - ID
  - Name
  - Class Name
  - Tag Name
  - CSS Selector
  - XPath (Absolute & Relative)

- Synchronization
  - Implicit Wait
  - Explicit Wait

- Web Element Interaction
  - Text Box
  - Button
  - Checkbox
  - Dropdown
  - Success Messages

---

## Tasks Completed

- Identified web elements using different locator strategies.
- Automated form input using Selenium WebDriver.
- Compared Absolute XPath and Relative XPath.
- Used CSS Selectors for efficient element identification.
- Implemented Implicit Wait for general synchronization.
- Implemented Explicit Wait using WebDriverWait and Expected Conditions.
- Validated success messages after user actions.
- Updated locators to match the latest LambdaTest Selenium Playground UI.
