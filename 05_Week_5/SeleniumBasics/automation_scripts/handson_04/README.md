# Selenium Basics - Hands-On 4

## Project Overview

This project is part of the Digital Nurture 5.0 QA Concepts & Selenium Basics Hands-On Exercises.

Hands-On 4 focuses on understanding Selenium architecture, setting up the Selenium WebDriver environment, and performing basic browser automation using Python.

---

## Tools and Technologies

- Python 3.10+
- Selenium
- webdriver-manager
- Google Chrome
- Visual Studio Code

## Installation

Install the required Python packages:

```bash
pip install -r requirements.txt
```

Or install them manually:

```bash
pip install selenium webdriver-manager
```

---

## Files

### setup_test.py

This script demonstrates:

- Selenium Architecture (WebDriver, Selenium Grid, Selenium IDE)
- Launching Chrome using webdriver-manager
- Opening the LambdaTest Selenium Playground
- Printing the page title
- Using an implicit wait
- Running Chrome in headless mode

### navigation_test.py

This script demonstrates:

- Navigating between pages
- URL validation using assertions
- Opening a new browser tab
- Switching between browser windows
- Capturing screenshots
- Getting and setting browser window size
