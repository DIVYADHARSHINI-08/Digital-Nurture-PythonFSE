# Handson 06 - React Student Portal

## Objective
Build a Student Portal application using React by implementing routing, state management, and component-based architecture.

## Technologies Used
- React
- Vite
- React Router DOM
- Redux Toolkit
- Context API
- JavaScript (ES6+)
- CSS

## Tasks Completed

### Task 1 – React Routing
- Configured React Router.
- Created navigation between pages.
- Implemented:
  - Home Page
  - Courses Page
  - Course Detail Page
  - Profile Page

### Task 2 – Context API
- Created `EnrollmentContext`.
- Managed enrolled course data globally.
- Shared state across multiple components without prop drilling.

### Task 3 – Redux Toolkit
- Configured Redux Store.
- Created `enrollmentSlice`.
- Implemented actions for:
  - Enroll Course
  - Remove Course
- Connected Redux Store with React components.
