# Handson 08 - Vue Student Portal

## Objective
Develop a Student Portal application using Vue.js by implementing components, routing, state management, and the Composition API.

## Technologies Used
- Vue.js 3
- Vite
- Vue Router
- Pinia
- JavaScript (ES6+)
- CSS

## Tasks Completed

### Task 1 – Vue Components and Routing
- Created reusable Vue components.
- Configured Vue Router.
- Implemented navigation between:
  - Home
  - Courses
  - Course Details
  - Profile

### Task 2 – State Management with Pinia
- Configured Pinia Store.
- Managed application state using Pinia.
- Implemented course enrollment functionality.
- Shared state across multiple components.

### Task 3 – Composition API
- Used Composition API features:
  - `ref()`
  - `computed()`
  - `defineProps()`
  - `defineEmits()`
  - `onMounted()`
- Built reactive and reusable components.
