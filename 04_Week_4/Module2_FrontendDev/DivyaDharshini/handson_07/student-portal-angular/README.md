# Handson 07 - Angular Student Portal

## Objective
Develop a Student Portal application using Angular by implementing components, routing, services, forms, dependency injection, and HTTP communication.

## Technologies Used
- Angular
- TypeScript
- Angular Router
- Angular Forms
- Reactive Forms
- Angular Services
- Dependency Injection
- HttpClient
- CSS

## Tasks Completed

### Task 1 – Angular Components and Routing
- Created standalone Angular components.
- Configured Angular Routing.
- Implemented navigation between:
  - Home
  - Courses
  - Course Details
  - Profile

### Task 2 – Forms and Services
- Implemented Template-driven and Reactive Forms.
- Used `ngModel` for two-way data binding.
- Created Angular Services for sharing data.
- Implemented Dependency Injection.

### Task 3 – HTTP Client Integration
- Configured HttpClient.
- Consumed REST APIs.
- Displayed data retrieved from API.
- Implemented basic error handling.
