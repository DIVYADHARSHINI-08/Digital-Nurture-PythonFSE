# Handson 09 - Web Accessibility and Responsive Design

## Objective
Improve the usability and accessibility of a web application by implementing accessibility best practices and creating a responsive user interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Accessibility (WCAG Principles)

## Tasks Completed

### Task 1 – Semantic HTML
- Used semantic HTML elements.
- Improved page structure for better accessibility.
- Added meaningful headings and landmarks.

### Task 2 – Accessibility Enhancements
- Added descriptive labels for form controls.
- Improved keyboard navigation.
- Included alternative text for images.
- Enhanced color contrast and readability.

### Task 3 – Responsive Design
- Applied responsive layouts using CSS.
- Optimized the application for different screen sizes.
- Improved overall user experience across desktop and mobile devices.
