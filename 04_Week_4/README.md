# Week 4 - Frontend Development

## Overview

This repository contains the solutions for **Week 4 – Frontend Development** as part of the **Cognizant Digital Nurture 4.0 - Python Full Stack Engineer (FSE)** program.

The hands-ons focus on modern frontend development using **React**, **Angular**, **Vue.js**, API integration, state management, and web accessibility.
