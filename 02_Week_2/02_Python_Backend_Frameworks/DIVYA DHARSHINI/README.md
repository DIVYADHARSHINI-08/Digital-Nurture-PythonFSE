# Python Backend Frameworks

## Module Overview

This module contains the hands-on exercises completed as part of the **Python Backend Frameworks** module in the **Cognizant Digital Nurture 5.0 – Python Full Stack Engineer Deep Skilling Program**.

The exercises focus on developing backend applications using three widely used Python frameworks:

- Django
- Flask
- FastAPI

Each hands-on builds practical skills in API development, database integration, authentication, and backend architecture.

---

## Objectives

The objectives of this module are to:

- Understand Python backend frameworks.
- Develop RESTful APIs.
- Perform CRUD operations.
- Work with ORM for database interaction.
- Integrate relational databases.
- Validate requests and responses.
- Implement authentication and authorization.
- Explore Microservices architecture.
- Build production-ready backend applications.
