This folder contains examples demonstrating the five SOLID principles in Python.

## Contents

| File | Principle |
|------|-----------|
| 02_SRP.py | Single Responsibility Principle |
| 03_OCP.py | Open/Closed Principle |
| 04_LSP.py | Liskov Substitution Principle |
| 05_ISP.py | Interface Segregation Principle |
| 06_DIP.py | Dependency Inversion Principle |
