# Week 1 - Software Engineering Fundamentals

## Overview

This repository contains the learning materials, sample programs, notes, hands-on exercises, and outputs completed during **Week 1** of the **Cognizant Digital Nurture 5.0 - Python Full Stack Engineer (FSE)** program.

The focus of this week is to build a strong foundation in software design, algorithms, database concepts, and unit testing.

---

## Modules Covered

### 1. Design Patterns and Principles
- SOLID Principles
- Creational Patterns
- Structural Patterns
- Behavioral Patterns
- Architectural Patterns (MVC & Dependency Injection)

### 2. Data Structures and Algorithms
- Analysis of Algorithms
- Arrays
- Searching Algorithms
- Sorting Algorithms

### 3. Database Integration
- SQL Concepts
- Database Design
- Hands-on Exercises *(To be completed)*

### 4. Unit Testing
- Testing Fundamentals
- PyTest
- unittest
- Jest
- Mocha
- Mocking
- Test-Driven Development (TDD)
- Code Coverage
