# Asymptotic Notation

## Big O (O)

Represents the upper bound or worst-case time complexity of an algorithm.

Example:
- Linear Search → O(n)
- Binary Search → O(log n)

---

## Big Omega (Ω)

Represents the best-case time complexity.

Example:
- Linear Search → Ω(1)

---

## Big Theta (Θ)

Represents the average or exact growth rate.

Example:
- Linear Search → Θ(n)

---

## Complexity Comparison

| Complexity | Performance |
|------------|-------------|
| O(1) | Constant |
| O(log n) | Logarithmic |
| O(n) | Linear |
| O(n log n) | Linearithmic |
| O(n²) | Quadratic |
