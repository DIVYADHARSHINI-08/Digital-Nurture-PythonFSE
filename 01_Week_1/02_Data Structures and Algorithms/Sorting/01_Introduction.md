# Sorting Algorithms

## Overview

Sorting is the process of arranging data in ascending or descending order. Efficient sorting algorithms improve searching performance and optimize data processing.

## Algorithms Covered

- Bubble Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

## Applications

- Database indexing
- Search optimization
- Data analysis
- Report generation

## Complexity Summary

| Algorithm | Best | Average | Worst |
|-----------|------|---------|-------|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Insertion Sort | O(n) | O(n²) | O(n²) |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |
| Heap Sort | O(n log n) | O(n log n) | O(n log n) |
