# Searching Algorithms

## Overview

Searching is the process of finding a specific element in a collection of data. Choosing the appropriate searching algorithm improves the performance of applications.

## Types Covered

- Linear Search
- Binary Search

## Comparison

| Algorithm | Time Complexity | Requirement |
|-----------|-----------------|-------------|
| Linear Search | O(n) | Works on any list |
| Binary Search | O(log n) | Requires sorted list |

## Applications

- Database lookup
- File searching
- Search engines
- Inventory management
